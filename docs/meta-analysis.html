A Reproducible Meta-Analysis in R

Author

1 Project Overview

This document presents a reproducible meta-analysis examining the effect of oral iron supplementation on haemoglobin (Hb) concentration in women of reproductive age (18–49 years). The analysis pipeline mirrors the approach I used in my published Cochrane abridged review on intravenous iron for non-anaemic iron deficiency (Dugan et al., 2022, J Cachexia Sarcopenia Muscle), adapted here for oral supplementation trials using a synthetic dataset for portfolio demonstration.

The full pipeline — from effect size calculation through to sensitivity analysis — is built in R using metafor and follows PRISMA 2020 reporting conventions.

1.1 Motivation

Iron deficiency disproportionately affects women of reproductive age due to menstrual iron losses, pregnancy demands, and inadequate dietary intake. Oral iron supplementation is the frontline intervention, but the magnitude of Hb response varies considerably across trials — driven by differences in baseline iron status, dose, duration, and formulation.

A meta-analysis lets us pool effect sizes for a more precise overall estimate, quantify heterogeneity to understand why results vary, test for publication bias, and identify moderators (dose, baseline status) that explain variation in response.

1.2 Statistical Framework

The random-effects model assumes the true effect size varies across studies:

\[\hat{\theta}_i = \theta_i + \varepsilon_i, \quad \theta_i \sim N(\mu, \tau^2), \quad \varepsilon_i \sim N(0, v_i)\]

Where \(\hat{\theta}_i\) is the observed effect in study \(i\), \(\mu\) is the overall mean effect, \(\tau^2\) is between-study variance, and \(v_i\) is within-study sampling variance. This two-level structure is almost always appropriate for clinical nutrition research where populations, protocols, and contexts vary meaningfully.

2.2 Computing Effect Sizes

Using raw mean differences (MD) since all studies report Hb in g/dL — no need to standardise.

\[\text{MD}_i = \bar{X}_{T,i} - \bar{X}_{C,i}, \quad v_i = \frac{s^2_{T,i}}{n_{T,i}} + \frac{s^2_{C,i}}{n_{C,i}}\]

3.2.1 Clinical Context

WHO defines anaemia in non-pregnant women as Hb < 12.0 g/dL. A pooled increase of ~0.71 g/dL is clinically meaningful — enough to shift a proportion of mildly anaemic women into the normal range. The wide prediction interval (explored below) tells a more nuanced story about who actually benefits.

A unimodal cluster suggests quantitative heterogeneity (studies sampling from one distribution). Multiple clusters would indicate distinct subpopulations — which is exactly what we test next.

6 Subgroup Analysis

6.1 By Baseline Iron Status

The key biological moderator here: iron-deficient women have suppressed hepcidin, so they absorb a greater fraction of supplemental iron and show larger Hb responses. This is the mechanism I investigated extensively in the IRONWOMAN trial.

7 Meta-Regression

7.1 Dose-Response Relationship

Meta-regression extends the RE model by including study-level covariates as predictors of effect size:

\[\theta_i = \beta_0 + \beta_1 \cdot \text{Dose}_i + u_i, \quad u_i \sim N(0, \tau^2_{\text{residual}})\]

7.3.1 Model Comparison

Code 
# R^2 analogue: proportion of between-study variance explained
# R^2 = 1 - (tau^2_model / tau^2_null)

# Helper for AICc extraction — row naming varies across metafor versions
get_aicc <- function(model) {
  fs <- fitstats(model)
  idx <- grep("AICc", rownames(fs), ignore.case = TRUE)
  if (length(idx) > 0) return(as.numeric(fs[idx[1], 1]))
  return(as.numeric(fs[4, 1]))
}

tibble(
  Model = c("Null (no moderators)", "Dose only", "Duration only",
            "Dose + Duration + Status"),
  tau2 = round(c(res$tau2, reg_dose$tau2, reg_dur$tau2, reg_full$tau2), 4),
  R2   = round(c(0,
                  max(0, 1 - reg_dose$tau2 / res$tau2),
                  max(0, 1 - reg_dur$tau2 / res$tau2),
                  max(0, 1 - reg_full$tau2 / res$tau2)) * 100, 1),
  AICc = round(c(get_aicc(res),
                  get_aicc(reg_dose),
                  get_aicc(reg_dur),
                  get_aicc(reg_full)), 1)
) %>%
  kable(
    col.names = c("Model", "\u03C4\u00B2 (residual)", "R\u00B2 (%)", "AICc"),
    caption = "Model comparison: How much heterogeneity do moderators explain?"
  ) %>%
  kable_styling(bootstrap_options = c("striped", "hover"), full_width = FALSE)

8.2 Egger’s Regression Test

Formalises the visual asymmetry assessment. A significant intercept (\(p < 0.10\) by convention) indicates funnel asymmetry — though this could reflect genuine small-study effects rather than publication bias specifically.

\[\frac{\hat{\theta}_i}{\text{SE}_i} = \beta_0 + \beta_1 \cdot \frac{1}{\text{SE}_i} + \varepsilon_i\]

## Project Overview

This document presents a reproducible meta-analysis examining the effect of oral iron supplementation on haemoglobin (Hb) concentration in women of reproductive age (18--49 years). The analysis pipeline mirrors the approach I used in my published Cochrane abridged review on intravenous iron for non-anaemic iron deficiency ([Dugan et al., 2022, *J Cachexia Sarcopenia Muscle*](https://doi.org/10.1002/jcsm.13075)), adapted here for oral supplementation trials using a synthetic dataset for portfolio demonstration.

The full pipeline — from effect size calculation through to sensitivity analysis — is built in R using `metafor` and follows PRISMA 2020 reporting conventions.

### Motivation

Iron deficiency disproportionately affects women of reproductive age due to menstrual iron losses, pregnancy demands, and inadequate dietary intake. Oral iron supplementation is the frontline intervention, but the magnitude of Hb response varies considerably across trials — driven by differences in baseline iron status, dose, duration, and formulation.

A meta-analysis lets us pool effect sizes for a more precise overall estimate, quantify heterogeneity to understand *why* results vary, test for publication bias, and identify moderators (dose, baseline status) that explain variation in response.

### Statistical Framework

The random-effects model assumes the true effect size varies across studies:

$$\hat{\theta}_i = \theta_i + \varepsilon_i, \quad \theta_i \sim N(\mu, \tau^2), \quad \varepsilon_i \sim N(0, v_i)$$

Where $\hat{\theta}_i$ is the observed effect in study $i$, $\mu$ is the overall mean effect, $\tau^2$ is between-study variance, and $v_i$ is within-study sampling variance. This two-level structure is almost always appropriate for clinical nutrition research where populations, protocols, and contexts vary meaningfully.

### Model Fitting


### Interpreting the Results

```{r extract-results}
pooled_md   <- round(coef(res), 2)
pooled_ci   <- round(res$ci.lb, 2)
pooled_ci_u <- round(res$ci.ub, 2)
tau2_val    <- round(res$tau2, 4)
I2_val      <- round(res$I2, 1)
Q_val       <- round(res$QE, 2)
Q_df        <- res$k - 1
Q_p         <- formatC(res$QEp, format = "e", digits = 2)
```

Key outputs:

- **Pooled MD** = `r pooled_md` g/dL (95% CI: `r pooled_ci` to `r pooled_ci_u`)
- **$\tau^2$** = `r tau2_val` — between-study variance beyond sampling error
- **$I^2$** = `r I2_val`% — proportion of variability due to genuine heterogeneity (>75% = high)
- **Cochran's Q** = `r Q_val` (df = `r Q_df`, $p$ = `r Q_p`)

#### Clinical Context

WHO defines anaemia in non-pregnant women as Hb < 12.0 g/dL. A pooled increase of ~`r pooled_md` g/dL is clinically meaningful — enough to shift a proportion of mildly anaemic women into the normal range. The wide prediction interval (explored below) tells a more nuanced story about who actually benefits.

## Meta-Regression

### Dose-Response Relationship

Meta-regression extends the RE model by including study-level covariates as predictors of effect size:

$$\theta_i = \beta_0 + \beta_1 \cdot \text{Dose}_i + u_i, \quad u_i \sim N(0, \tau^2_{\text{residual}})$$

```{r meta-regression-dose}
# Testing dose-response — biologically plausible up to ~100mg,
# beyond which mucosal block limits absorption

summary(reg_dose)
```

```{r model-comparison}
  idx <- grep("AICc", rownames(fs), ignore.case = TRUE)

  return(as.numeric(fs[4, 1]))
}
```

### Egger's Regression Test

Formalises the visual asymmetry assessment. A significant intercept ($p < 0.10$ by convention) indicates funnel asymmetry — though this could reflect genuine small-study effects rather than publication bias specifically.

$$\frac{\hat{\theta}_i}{\text{SE}_i} = \beta_0 + \beta_1 \cdot \frac{1}{\text{SE}_i} + \varepsilon_i$$

```{r eggers-test}
egg <- regtest(res, model = "lm")
egg
```

### Trim-and-Fill Method

Estimates the number of "missing" studies and imputes them for an adjusted pooled estimate.

```{r trim-and-fill, fig.height=7}
tf <- trimfill(res)
summary(tf)

funnel(tf,
       xlab    = "Mean Difference (g/dL)",
       main    = "Trim-and-Fill: Adjusted Funnel Plot",
       col     = "steelblue",
       bg      = adjustcolor("steelblue", alpha = 0.4),
       refline = coef(tf),
       legend  = TRUE)
```

Trim-and-fill assumes asymmetry is entirely due to publication bias, which isn't always true — small-study effects can arise from genuine biological reasons (e.g., smaller trials may select more deficient populations or use higher doses). I treat the adjusted estimate as a sensitivity check, not a corrected truth.

## Sensitivity & Influence Analysis

### Leave-One-Out Analysis

Refits the model $k$ times, each time dropping one study. If removing a single study shifts the pooled estimate substantially, the result isn't robust.

```{r leave-one-out, fig.height=9}
loo <- leave1out(res)

par(mar = c(5, 10, 4, 2))
forest(
  x      = loo$estimate,
  ci.lb  = loo$ci.lb,
  ci.ub  = loo$ci.ub,
  slab   = dat$author,
  xlab   = "Pooled MD (g/dL) with Study Removed",
  main   = "Leave-One-Out Sensitivity Analysis",
  refline = coef(res),
  cex    = 0.8,
  header = "Study Removed"
)
```

### Influence Diagnostics

```{r influence-diagnostics, fig.height=10}
inf <- influence(res)
plot(inf, plotdfb = TRUE)
```

Key thresholds: studentised residuals > |2| flag outliers; Cook's distance identifies disproportionate influence; hat values > $2p/k$ indicate high leverage. In my experience with iron supplementation meta-analyses, outlier studies usually trace back to methodological differences — different formulations, co-interventions, or population characteristics.
